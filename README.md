# 个人博客网站

这是一个使用 Vite + React + TypeScript 构建的现代化个人博客网站。

## 功能特点

- 响应式设计，适配各种设备
- 博客文章列表展示
- 文章详情页面
- 文章分类和标签系统
- 暗色/亮色主题切换
- 评论系统
- 搜索功能
- 个人信息展示

## 技术栈

- Vite - 构建工具
- React - 前端框架
- TypeScript - 类型系统
- TailwindCSS - 样式框架
- React Router - 路由管理
- Markdown-it - Markdown 渲染
- zustand - 状态管理
